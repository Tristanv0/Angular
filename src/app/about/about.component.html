Experience and Challenges Implementing Angular

How I Installed The Framework

Before creating the repository, I had to install git and Node.js. Then, I followed the sequence of commands:

# Install Angular CLI globally
npm install -g @angular/cli

# Verify that Angular CLI has been installed successfully
ng version

# Clone this repository
git clone https://github.com/Tristanv0/Angular.git

# Go into the repository
cd Angular

# Run the following command to start the Angular development server
ng serve

# You can use Ctrl+C to end the development server

Challenges I Encountered

The Angular framework is mainly made up of TypeScript. As someone who has never learned or used TypeScript, this was a big challenge for me to overcome. I also had to watch several videos on how to set up the framework as many of the videos out there are outdated. I was able to overcome this challenge from reviewing the documentation. Although reading the documentation was a longer process than simply watching a video, I was able to successfully implement the framework.
